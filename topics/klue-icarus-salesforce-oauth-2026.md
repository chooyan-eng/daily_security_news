# Klue OAuth侵害と「Icarus」キャンペーン（2026年6月）

## 概要

競合インテリジェンスプラットフォームKlueのバックエンドへの侵害を起点とし、脅威アクター「Icarus」グループが多数の企業のSalesforce CRMデータを窃取した攻撃キャンペーン。2026年6月11日にKlueのバックエンドインフラへ不正アクセスし、OAuthトークンを収集する悪意あるコードを展開した。Salesforceは6月19日にKlue Battlecardsアプリの接続を全顧客環境で無効化した。

**同一性の判断に役立つ情報：**
- 脅威アクター: Icarus（2026年4月下旬から活動）
- 侵害起点: Klue（競合インテリジェンスプラットフォーム）
- 手法: OAuth token harvesting via compromised third-party integration
- 影響サービス: Salesforce、HubSpot、SharePoint、Zoom、Gong、Google Drive、Slack
- 帰属根拠: Session Messenger IDの一致（恐喝メール / ダークウェブリークサイト）
- 対応組織: Huntress、ReliaQuest、Salesforce
