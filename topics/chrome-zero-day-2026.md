# Chrome V8 ゼロデイ脆弱性シリーズ（2026年）

## 概要

2026年に入り、Google Chrome の V8 JavaScript/WebAssembly エンジンを含む複数のゼロデイ脆弱性が実際に悪用されている。CVE-2026-11645 は6月8日リリースの Chrome 149（バージョン149.0.7827.103以降）で修正された境界外読み書き（OOB Read/Write）脆弱性で、細工された HTML ページを開くだけでサンドボックス内でのRCEが可能。2026年に確認された5例目の悪用済みChromeゼロデイ。

**同一性の判断に役立つ情報：**
- CVE（最新）: CVE-2026-11645
- 対象コンポーネント: Chrome V8 JavaScript/WebAssembly エンジン
- 修正バージョン: 149.0.7827.103以降
- 悪用確認: Googleが実際の悪用を確認済み
