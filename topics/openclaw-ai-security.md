# OpenClaw AIエージェントのセキュリティ脆弱性

## 概要

人気の自己ホスト型AIエージェントプラットフォーム「OpenClaw」に複数の深刻なセキュリティ脆弱性が発見されている。間接プロンプトインジェクション、ローカルインスタンスのハイジャック（ClawJacked）、ClawHub マーケットプレイスの悪意あるスキル（最大12%が悪性）など。根本原因はLLMがデータと命令を同一コンテキストで処理するという構造的問題であり、完全な解決策が存在しない。

**同一性の判断に役立つ情報：**
- 対象製品: OpenClaw（自己ホスト型AIエージェント）
- ClawJacked脆弱性パッチ: バージョン2026.2.26（2026-02-26リリース）
- 発見組織: Oasis Security（ClawJacked）、Imperva（プロンプトインジェクション）
