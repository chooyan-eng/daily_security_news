# FBI/CISA：ロシア諜報機関によるSignalバックアップリカバリーキー窃取フィッシング（2026年6月）

## 概要

FBI・CISAが2026年6月26日に発出した更新済み公式勧告（PSA I-062626-PSA）により、ロシア諜報機関（RIS）に紐づく脅威アクターグループ UNC5792・UNC4221 がSignalユーザーへのフィッシングキャンペーンを進化させ、バックアップリカバリーキーの窃取に手口を拡大したことが明らかになった。

バックアップリカバリーキーを入手した攻撃者は、被害者の過去のすべてのメッセージ履歴を攻撃者端末に復元できる。さらに「同じ電話番号で新しいSignalアカウントを作成しても旧キーは無効化されない」という致命的な特性がある。

**同一性の判断に役立つ情報：**
- PSA番号: PSA I-062626-PSA
- 帰属: UNC5792・UNC4221（ロシア諜報機関RIS、FSB Border Guards組み込み・GRU関連）
- 前回勧告: 2026年3月（Signalアカウント乗っ取りフィッシング）
- 攻撃手口: Signalサポートをかたるフィッシングでリカバリーキーをコピーさせる
- 主要標的: 米国・国際政府関係者、軍人、政治家、ジャーナリスト、ウクライナ在住関係者
