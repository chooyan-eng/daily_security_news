# Microsoft Patch Tuesday 2026年6月

## 概要

2026年6月10日リリースのMicrosoft月例セキュリティ更新プログラム。単月として過去最多の206件の脆弱性を修正し、うち39件がCritical、3件が悪用実績のあるゼロデイ。Windows Kernel RCE（CVE-2026-45657、CVSS 9.8・ワーム可能）、Remote Desktop Client（11件のCVE）、Hyper-V VM脱出、Office RCEなど広範な製品群に深刻な脆弱性が含まれる。

**同一性の判断に役立つ情報：**
- 更新プログラムリリース日: 2026-06-10
- 主要CVE: CVE-2026-45657（Windows Kernel RCE、CVSS 9.8）、CVE-2026-44801、CVE-2026-47652 他
- 修正総数: 206件（過去最多）
- Criticalの件数: 39件
