# 北朝鮮 APT37（ScarCruft）NarwhalRAT スピアフィッシングキャンペーン（2026年6月）

## 概要

北朝鮮の国家支援型ハッカーグループ ScarCruft（APT37）が2026年6月に観測された新しいキャンペーン。Microsoft アカウントセキュリティ通知を偽装したスピアフィッシングメールで新型マルウェア「NarwhalRAT」を配布。従来の主力マルウェア RokRAT から NarwhalRAT への転換が確認されており、キーロギング・スクリーンショット・音声録音機能を持ち pCloud や韓国系ウェブサイトを C2 チャネルとして利用する。

**同一性の判断に役立つ情報：**
- 脅威アクター: ScarCruft / APT37（北朝鮮帰属）
- マルウェア: NarwhalRAT（RokRAT からの転換）
- 配布手法: 偽 Microsoft セキュリティ通知 → ZIP → LNK ファイル
- C2 チャネル: pCloud（デッドドロップ）、韓国系ウェブサイト
- 主要ターゲット: 韓国政府・防衛・研究機関、Microsoft 365 使用組織
