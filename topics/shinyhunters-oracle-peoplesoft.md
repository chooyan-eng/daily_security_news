# ShinyHunters による Oracle PeopleSoft 攻撃キャンペーン

## 概要

脅威アクター ShinyHunters（Mandiant は UNC6240 として追跡）が、Oracle PeopleSoft の未パッチゼロデイ脆弱性 CVE-2026-35273（CVSS 9.8）を悪用し、大学を中心とした100社以上の組織に侵入したキャンペーン。2026年5月27日〜6月9日にかけて活動が確認された。Oracle は6月10日にパッチを公開。

**同一性の判断に役立つ情報：**
- CVE: CVE-2026-35273
- 脅威アクター: ShinyHunters / UNC6240
- 対象製品: Oracle PeopleSoft Enterprise PeopleTools
- 攻撃期間: 2026-05-27〜2026-06-09
