# ロシア系APT による Signal バックアップキー窃取キャンペーン（UNC5792/UNC4221）

## 概要

ロシア諜報機関系ハッカーグループ（UNC5792・UNC4221）が Signal ユーザーを標的にしたフィッシングキャンペーンで、2026年6月に FBI・CISA が更新版の警告を発出。新たな手口では Signal のバックアップリカバリーキーを盗むことで、攻撃者が過去のすべてのメッセージ履歴・メディアファイルにアクセス可能となる。Signal サポートを装ったソーシャルエンジニアリング経由でキーを窃取する。2026年3月初期勧告の発展版として PSA I-062626-PSA として公表。

**同一性の判断に役立つ情報：**
- 脅威アクター: UNC5792（FSB系）、UNC4221（ロシア軍諜報系）
- 標的: Signal ユーザー（政府・軍・政治・ジャーナリスト・ウクライナ関係者）
- 攻撃手法: Signal サポートを装ったソーシャルエンジニアリングでリカバリーキーを窃取
- FBI/CISA 警告番号: PSA I-062626-PSA（2026年6月26日発出）
- 旧手口: アカウント PIN・認証コード窃取、デバイスリンク
- 新手口（2026年6月〜）: Backup Recovery Key の窃取
- 危険性: 窃取したキーは新アカウント作成後も有効
