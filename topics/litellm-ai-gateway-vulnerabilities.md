# LiteLLM AIゲートウェイ脆弱性チェーン

## 概要

オープンソースのAIゲートウェイ「LiteLLM」に発見された複数の脆弱性チェーン。低権限ユーザーが管理者権限に昇格し、サーバー上で任意コードを実行できる（CVSS 最大 9.8）。OpenAI・Anthropicなど100以上のモデルプロバイダーへの統合ブローカーとして多数の企業がデプロイしているため、侵害成功時にはすべてのAI APIキーの窃取・AI呼び出しのハイジャックが可能。

**同一性の判断に役立つ情報：**
- CVE-2026-47101: APIキー生成時の権限チェック不備（CVSS 8.8）
- CVE-2026-47102: フィールドレベル認可の欠如による権限昇格（CVSS 7.5）
- CVE-2026-40217: Python exec()サンドボックスエスケープ→RCE（CVSS 9.8）
- CVE-2026-42271: MCPテストエンドポイントのコマンドインジェクション（CVSS 8.7）— CISAがKEVに追加（2026-06-08）
- CVE-2026-48710: StarletteのHTTPホストヘッダーバイパス（未認証RCEに連鎖）
- パッチ済みバージョン: 1.83.14-stable以降
- CISA KEV修正期限: 2026-06-22
