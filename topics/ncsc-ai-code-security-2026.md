# 英国NCSCのAIコード生成セキュリティリスク警告（2026年）

## 概要

英国国家サイバーセキュリティセンター（NCSC）が2026年に発出した、AI生成コードのセキュリティリスクに関する一連の警告・ガイダンス。AIコーディングアシスタントが生成したコードを適切なレビューなしに採用する「バイブコーディング（vibe coding）」スタイルが、セキュリティ脆弱性の混入・監査困難なコードベースの生成・過度な信頼といったリスクをもたらすと指摘。クリティカルシステムにはリスクレベルに応じた徹底的なテスト・監査・レビューを求めている。

**同一性の判断に役立つ情報：**
- 発行機関: 英国 NCSC（National Cyber Security Centre）
- 警告対象: AIコーディングアシスタント全般（Claude Code、GitHub Copilot、Cursor 等）
- 関連概念: バイブコーディング（vibe coding）、AI脆弱性発見
- 関連脆弱性: Claude Code CVE-2025-59536、CVE-2026-21852 等（AIコーディングツール自体の脆弱性）
