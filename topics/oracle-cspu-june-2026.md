# Oracle Critical Security Patch Update 2026年6月

## 概要

Oracle が2026年6月16〜17日にリリースした Critical Security Patch Update（CSPU）。245件の新しいセキュリティパッチを複数の製品ファミリーに適用し、うち10件がクリティカル評価。リモートから認証なしに悪用可能な脆弱性が約100件含まれる。Oracle Fusion Middleware に最多（100件超）のパッチが集中。ShinyHunters に積極的に悪用されていた PeopleSoft の CVE-2026-35273（CVSS 9.8）の恒久修正も含まれる。

**同一性の判断に役立つ情報：**
- 更新種別: Oracle Critical Security Patch Update（CSPU）June 2026
- 公開日: 2026-06-16〜17
- 総パッチ数: 245件
- クリティカル: 約10件
- 注目CVE: CVE-2026-35273（PeopleSoft、CVSS 9.8、ShinyHunters 悪用済み）
- 最大影響製品: Oracle Fusion Middleware（100件超）
