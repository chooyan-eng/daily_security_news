# Ubiquiti UniFi OS 最大深刻度脆弱性の悪用（2026年6月）

## 概要

Ubiquiti UniFi OS Server に存在するアクセス制御・パストラバーサル・コマンドインジェクションの3つの最大深刻度脆弱性（CVE-2026-34908/34909/34910）が 2026年6月に積極的に悪用されていることが確認された。CISA KEV への追加と連邦機関向け 6月15日修正期限が設定された。通知を受けた組織の 68% が大学・大学院などの高等教育機関であり、ネットワーク管理インフラへの攻撃が特徴。

**同一性の判断に役立つ情報：**
- CVE: CVE-2026-34908（不適切なアクセス制御）、CVE-2026-34909（パストラバーサル）、CVE-2026-34910（コマンドインジェクション）
- 影響製品: Ubiquiti UniFi OS Server（Dream Machine、Dream Router、Cloud Gateway 等）
- 深刻度: 最大（全3件がCritical）
- 攻撃パターン: 3脆弱性の連鎖によるリモート root RCE
- CISA KEV 追加: 2026年6月初旬
- 連邦機関修正期限: 2026年6月15日（期限到来済み）
- 主な標的業種: 高等教育機関（通知組織の68%）
