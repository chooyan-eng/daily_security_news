# FBI・CISA警告：ロシア諜報機関がSignalバックアップリカバリーキーを狙うフィッシングに手口を進化

- **日付**: 2026-06-27
- **出典**: [BleepingComputer](https://www.bleepingcomputer.com/news/security/fbi-russian-hackers-now-target-signal-backup-recovery-keys/) / [Hacker News](https://thehackernews.com/2026/06/fbi-warns-russian-intelligence-hackers.html) / [The Next Web](https://thenextweb.com/news/fbi-russian-hackers-signal-backup-recovery-key-unc5792)
- **トピック**: [ロシア系APT による Signal バックアップキー窃取キャンペーン（UNC5792/UNC4221）](../topics/signal-backup-key-russian-phishing-2026.md)
- **分類**: 新規

## 概要

FBI と CISA が2026年6月26日、ロシア諜報機関系ハッカーグループ（UNC5792・UNC4221）が Signal ユーザーを狙うフィッシングキャンペーンを更新した警告（PSA I-062626-PSA）を発出した。新たな手口では Signal の「バックアップリカバリーキー」を標的とし、奪取に成功すれば攻撃者が過去のメッセージ履歴すべてにアクセス可能となる。

## 詳細

### キャンペーンの変遷

本警告は2026年3月に発出された初期勧告の更新版で、攻撃者の戦術転換を報告している。

- **旧手口**: Signal の認証コード・アカウント PIN の窃取、または攻撃者デバイスをリンクさせる
- **新手口**: **バックアップリカバリーキー**（Backup Recovery Key）の窃取

リカバリーキーを取得することで、攻撃者は自分のデバイスにバックアップを復元し、被害者の過去のすべての Signal メッセージ・グループチャット・メディアにアクセスできる。

### フィッシングの具体的な手口

**第1段階**: Signal サポートチームを装ったメッセージが届く
> 「イランおよびポスト・ソ連諸国のハッカーによる攻撃が相次いでいるため、Signal が必須の二段階認証を導入します」

**第2段階**: 「同期の問題」を警告する第2のメッセージが届く
> 「あなたの Signal アカウントデータ（メッセージとメディア）が同期の問題により永久に失われる危険があります。データ損失を防ぐため、バックアップ設定でリカバリーキーをクリップボードにコピーし、このメッセージに貼り付けてください」

### 危険な理由

- Signal でアカウントを新規作成しても（同一番号での再登録）、窃取されたリカバリーキーは**失効しない**
- 攻撃者は窃取したキーで過去メッセージに**継続的にアクセス**可能
- エンドツーエンド暗号化そのものは破られていないが、バックアップ復号が可能になる

### 脅威アクターの帰属

FBI は以下のグループを関連付けて特定：
- **UNC5792**: FSB（ロシア連邦保安局）の国境警備隊に組み込まれた FSB 職員
- **UNC4221**: ロシア軍諜報機関の関連グループ

### 主な標的

現職・元職の米国および国際的な政府高官、軍関係者、政治家、ジャーナリスト、ウクライナに所在する重要人物

### 推奨対処

1. Signal の「バックアップリカバリーキー」を誰にも共有しない
2. Signal サポートを名乗るメッセージには警戒する
3. デバイスのリンク一覧を定期的に確認し、不審なデバイスを削除する
