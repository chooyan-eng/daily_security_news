# Mastra AIフレームワーク npm 144パッケージがサプライチェーン攻撃で侵害：週間140万DLに影響

- **日付**: 2026-06-17
- **出典**: [Socket.dev](https://socket.dev/blog/mastra-npm-packages-compromised) / [CybersecurityNews](https://cybersecuritynews.com/140-mastra-npm-packages-compromised/) / [StepSecurity](https://www.stepsecurity.io/blog/mastra-npm-packages-compromised-using-easy-day-js)
- **トピック**: [Mastra npm サプライチェーン攻撃（2026年6月）](../topics/mastra-npm-supply-chain-2026.md)
- **分類**: 新規

## 概要

2026年6月17日、AI フレームワーク Mastra の npm 組織（@mastra スコープ）が侵害され、144パッケージに情報窃取マルウェアを含む悪意ある依存関係が注入された。`dayjs` のタイポスクワット `easy-day-js` を媒介として88分間で全パッケージが再公開され、週間110万以上のダウンロードを持つ `@mastra/core` を含む広範なエコシステムが影響を受けた。

## 詳細

### 攻撃のタイムライン

| 時刻（UTC） | イベント |
|-------------|---------|
| 2026-06-16 07:05 | npm ユーザー `sergey2016` が `easy-day-js@1.11.21` を公開（クリーン版、信頼確立用） |
| 2026-06-17 | @mastra 組織の認証情報が侵害される |
| 攻撃開始 +11分 | 自動化スクリプトが144パッケージの再公開を開始 |
| 攻撃開始 +88分 | 全144パッケージの再公開完了 |
| 公開から6分後 | Socket がマルウェアを検出・警告 |

### 攻撃手法：タイポスクワット + 組織アカウント侵害

**ステップ1：信頼の確立（easy-day-js）**

`easy-day-js` は人気日付ライブラリ `dayjs`（週間数百万DL）のタイポスクワット。攻撃者は以下の点を完全に模倣し、視覚的検査をパスできるよう偽装した：
- バージョン番号（1.11.x系）
- 作者メタデータ（iamkun）
- ホームページ・リポジトリ URL
- ライセンスとキーワード

最初に公開された版は完全にクリーンで機能的なコピーであり、信頼性の確立だけを目的としていた。

**ステップ2：悪意あるバージョンの公開**

@mastra 組織の認証情報を侵害した後、攻撃者は自動化スクリプトで `easy-day-js`（悪意あるペイロード入り）を `@mastra/core` 等144パッケージの依存関係として追加し再公開した。

悪意あるペイロード：
- `postinstall` フックで実行される難読化されたドロッパー
- 攻撃者管理サーバから第二段階ペイロードをダウンロード
- 実行後に自身を削除して痕跡を消去

### マルウェアの機能

展開されたインフォスティーラーは以下を窃取：
- 暗号資産ウォレットデータ
- ブラウザ履歴・保存済み認証情報
- 開発者認証情報（API キー、SSH キー等）

クロスプラットフォーム対応（Windows / macOS / Linux）。

### 影響範囲

- 影響パッケージ数: 144パッケージ
- 最大被害パッケージ: `@mastra/core`（週間918,000以上のダウンロード）
- 合計週間ダウンロード: 110万以上
- ターゲット: AI フレームワークを利用する開発者（開発者認証情報を直接狙う）

### 対応

Mastra プロジェクトは侵害を受け、影響を受けたすべてのパッケージのクリーンなバージョンを再公開した。

### 推奨対応

- `@mastra/` スコープパッケージを最新の安全版に更新
- npm 監査（`npm audit`）を実施
- 開発環境上の認証情報（API キー、SSH キー、暗号資産ウォレット）を総点検・ローテーション
- `easy-day-js` が依存関係に含まれていないか確認・削除
- CI/CD 環境のシークレットを即時ローテーション
