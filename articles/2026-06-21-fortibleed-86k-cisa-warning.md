# FortiBleed続報 — 侵害デバイス86,644台に拡大、CISA が緊急警告・デフォルトアカウント悪用が主因

- **日付**: 2026-06-21
- **出典**: [BleepingComputer](https://www.bleepingcomputer.com/news/security/cisa-warns-fortinet-users-to-secure-devices-after-fortibleed-leak/)
- **トピック**: [FortiBleed：Fortinet Firewall 大規模認証情報流出キャンペーン（2026年6月）](../topics/fortibleed-fortinet-credential-harvesting.md)
- **分類**: 続報

## 概要

CISA が Fortinet FortiGate ユーザーに緊急警告を発出し、「FortiBleed」として知られる認証情報流出キャンペーンにより86,644台のファイアウォール・VPN の認証情報が漏洩したことを報告した。侵害された認証情報のうち汎用管理者アカウント（35%）とFortinet 組み込みシステムアカウント（28.3%）が過半数を占め、デフォルト認証情報の変更・ファクトリー認証情報のローテーションを怠ったことが主因であることが判明した。

## 詳細

### 最新の侵害状況

| 指標 | 数値 |
|------|------|
| 確認済み侵害デバイス数 | 86,644台 |
| 汎用管理者アカウント | 35%（最多） |
| Fortinet 組み込みシステムアカウント | 28.3% |
| 侵害国数 | 194カ国（前回同様） |

### CISA の主要勧告

CISA は FortiGate ユーザーに対し以下を即時実施するよう要請：

1. **デフォルトアカウントの名称変更または削除**: `admin` や Fortinet 組み込みアカウントはすべて変更・無効化
2. **ファクトリー認証情報のローテーション**: 工場出荷時デフォルトパスワードを使用し続けているデバイスは即時変更
3. **最新ファームウェアへの更新**: 特に CVE-2022-40684 を含む既知の脆弱性を修正済みバージョンへ
4. **侵害インジケーターの確認**: CISA が公開した IoC リストと照合し、侵害の有無を検証
5. **VPN アクセスログの精査**: 不審なログインや設定変更を遡及調査

### 侵害メカニズムの詳細

FortiBleed のロシア語話者脅威アクターは以下の多層アプローチを組み合わせた：

- **CVE-2022-40684 の遅延パッチ適用を悪用**: 2022年10月にパッチが公開されたにもかかわらず適用されていないデバイスへ認証バイパスで侵入
- **デフォルト認証情報の総当たり**: 変更されていないデフォルトアカウントへのパスワードスプレー
- **既侵害デバイスからの横展開**: 侵害済みデバイスを踏み台に、内部ネットワーク上の他のFortiGateデバイスの認証情報を自動収集
