## 概要

Microsoft は 2026年2月から活動する Windows ベースの暗号通貨クリッパーマルウェアキャンペーン「CryptoBandits」の詳細を公開した。USB ドライブ経由で拡散する LNK ワームを配信媒体とし、Tor 匿名ネットワークを用いた C2 通信で検出を回避。クリップボードの暗号通貨ウォレットアドレスを置き換えるだけでなく、スクリーンショット窃取やリモートコード実行機能も備えたバックドアへと進化している。

## 詳細

### キャンペーン概要

Microsoft Threat Intelligence は2026年6月17日、「CryptoBandits」と命名した Windows ベースの暗号通貨クリッパーキャンペーンの詳細を公開した。このキャンペーンは少なくとも2026年2月から活動が確認されており、主に個人暗号通貨ユーザーと開発者を標的にしている。

### 感染経路：USB LNK ワーム

マルウェアの主要な配布方法は USB ドライブを介した Windows ショートカット（.LNK）ファイルである：

1. 感染した USB デバイスに悪意ある LNK ファイルが作成される
2. 被害者が LNK ファイルを開くと、ワームコンポーネントが起動する
3. ワームは対象マシンが既に感染済みかを確認し、未感染の場合のみリモートサーバーからペイロードを取得する
4. 感染後は接続された新たな USB ドライブにも自己複製して拡散する

### 技術的機能

CryptoBandits の主要な機能：

**クリップボード操作**:
- クリップボード内容を10秒ごとに監視
- Bitcoin・Ethereum ウォレットアドレスを攻撃者制御のアドレスに置き換える
- BIP39 ニーモニックフレーズ（シードフレーズ）を検索・窃取する
- 秘密鍵の文字列パターンを検出してスキャンする

**情報窃取**:
- スクリーンショットを定期的に撮影してC2サーバーへ送信する
- ブラウザ保存のウォレット関連データを収集する

**Tor ベース C2 通信**:
- ポータブル Tor クライアントをバンドルして展開する
- ローカル SOCKS5 プロキシ経由で Tor 隠しサービス（.onion）のC2に通信する
- これにより通信先 IP を隠蔽し、標準的なネットワーク監視をすり抜ける

**バックドア機能**:
- リモートコード実行（RCE）機能を内蔵し、金融目的のステーラーから軽量バックドアへと進化
- Windows Script Host と ActiveX を利用した持続化メカニズム

### 実装技術

- Windows Script Host（WSH）と ActiveX を活用した起動ロジック
- Tor プロキシとの SOCKS5 インターフェース実装
- クリップボード API の高頻度ポーリング（10秒間隔）

### 対策

1. 信頼できない USB デバイスの接続禁止（AutoPlay の無効化）
2. Windows Script Host（WSH）の無効化または制限
3. 暗号通貨取引時のウォレットアドレスのコピー&ペースト後の目視確認
4. ハードウェアウォレット使用によるクリップボード経由攻撃の回避
5. エンドポイント保護ソリューションによる LNK ファイルの検査
