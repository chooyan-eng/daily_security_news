# Mastra npm サプライチェーン攻撃続報 — Microsoft が北朝鮮 Sapphire Sleet（BlueNoroff）への帰属を確認

- **日付**: 2026-06-21
- **出典**: [BleepingComputer](https://www.bleepingcomputer.com/news/security/)
- **トピック**: [Mastra npm サプライチェーン攻撃（2026年6月）](../topics/mastra-npm-supply-chain-2026.md)
- **分類**: 続報

## 概要

2026年6月17日に発生した Mastra AI フレームワーク npm パッケージへのサプライチェーン攻撃について、Microsoft の脅威インテリジェンスチームが北朝鮮の国家支援型脅威アクター「Sapphire Sleet」（別名 BlueNoroff）の関与を確認・公表した。144パッケージへのマルウェア注入を88分で実行したこの攻撃は、暗号資産窃取を専門とする北朝鮮組織が開発者サプライチェーンへ活動範囲を拡大した事例として注目される。

## 詳細

### Sapphire Sleet（BlueNoroff）について

Sapphire Sleet は北朝鮮政府に帰属する脅威アクターで、以下の活動で知られる：

- **主要任務**: 暗号資産・金融資産の窃取（北朝鮮の制裁回避のための外貨獲得）
- **別名**: BlueNoroff、TA444、CryptoCore
- **過去の主要標的**: 暗号取引所、DeFi プロトコル、ブロックチェーン企業、VC 投資家
- **帰属国**: 北朝鮮（Lazarus Group の傘下組織とされる）

### Mastra 攻撃への帰属の根拠

Microsoft が帰属の根拠として挙げたとされる要素：

1. **マルウェアのコードシグネチャ**: Sapphire Sleet が過去キャンペーンで使用したスティーラーと同一のコードパターン
2. **インフラの重複**: C2 サーバーと暗号資産ウォレットアドレスが既知の Sapphire Sleet インフラと一致
3. **攻撃の目的**: 開発者の暗号資産ウォレット・API キー窃取という Sapphire Sleet の典型的な動機と合致
4. **タイムゾーンと運用パターン**: 攻撃の実行タイミングが北朝鮮の業務時間帯に一致

### 攻撃の影響の再評価

| 指標 | 詳細 |
|------|------|
| 侵害されたパッケージ数 | 144パッケージ（@mastra スコープ全体） |
| 最大被害パッケージ | @mastra/core（週間918,000 DL） |
| 攻撃実行時間 | 88分 |
| 窃取対象データ | 暗号資産ウォレット秘密鍵・ブラウザ認証情報・API キー |

Sapphire Sleet の主要標的が暗号資産であることから、侵害されたマシン上の暗号資産関連ファイル（`.wallet`、`keystore.json`、シードフレーズを含むファイル等）が集中的に探索されたとみられる。

### 波及的な懸念

Sapphire Sleet が npm サプライチェーン攻撃に参入したことは、従来の金融機関・暗号取引所への直接標的型攻撃から**開発者エコシステムへのサプライチェーン攻撃**へ戦術を拡大していることを示す。AI フレームワーク開発者という高価値ターゲットを通じて、多数の企業のクラウドインフラや暗号資産保有を一度に狙う高効率な手法。

### 対応状況

- npm は問題のパッケージを削除済み（発覚から6分後に Socket が検出、即時対応）
- @mastra の正規パッケージはクリーンなバージョンに置き換え済み
- 影響期間中にインストールした開発者はシークレット全体（暗号資産ウォレット・API キー）のローテーションを強く推奨
