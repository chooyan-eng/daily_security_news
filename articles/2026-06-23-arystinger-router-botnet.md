# AryStinger ボットネット：4,300台以上の D-Link ルーターを偵察・プロキシネットワークに悪用

- **日付**: 2026-06-23
- **出典**: [The Hacker News](https://thehackernews.com/2026/06/arystinger-malware-infects-4300-legacy.html)
- **トピック**: [AryStinger ボットネット（D-Link ルーター大規模感染、2026年6月）](../topics/arystinger-router-botnet-2026.md)
- **分類**: 新規

## 概要

未発見のマルウェア「AryStinger」が4,300台以上のレガシー D-Link ルーター（主にDIR-850L/DIR-818LW）に感染し、偵察・プロキシネットワークを構築していることが明らかになった。13年前の既知の CVE を悪用し、感染ルーターを分散偵察エグゼキューターとして使用。DNS 設定の改ざんや全ネットワークトラフィックの傍受が可能。感染の約48%が韓国、32%が中国に集中。

## 詳細

### AryStinger の概要

AryStinger は以前未発見だったマルウェアファミリーで、Malwarebytes 研究者によって発見された。対象は EOL（サポート終了）を迎えた D-Link ルーターおよび NAS デバイスで、これらを攻撃インフラの「エグゼキューター」に変換する。

### 悪用する脆弱性

AryStinger は以下の既知の古い CVE を悪用：

- **CVE-2013-3307**（13年前）
- **CVE-2016-5681**（10年前）
- **CVE-2025-11837**（比較的新しい）

D-Link DIR-850L と DIR-818LW は EOL 製品であり、これらの CVE に対するベンダーパッチが提供されていないため、感染したデバイスを保護する手段が限定される。

### 主要な機能

感染したルーターは以下の機能を攻撃者に提供する：

1. **分散偵察**：コントローラーが多数のエグゼキューターにスキャンジョブを割り当て、並列でネットワーク・脆弱サービスのマッピングを実施
2. **プロキシ・トラフィックトンネリング**：攻撃者のトラフィックを感染ルーター経由で匿名化・中継
3. **DNS 改ざん**：被害者のブラウザトラフィックをフィッシングページやマルウェアホスティングサイトへリダイレクト
4. **トラフィック傍受**：ルーターを通過するすべての受信・送信トラフィックの監視と窃取
5. **コマンド実行**：感染デバイス上での任意コマンド実行

### マルウェアのバリアント

研究者は2種類のバリアントを発見：

- **C 言語版**：レガシーベースのルーター（DIR-850L、DIR-818LW 等）をターゲット。感染台数が多い
- **Go 言語版**：NAS デバイスをターゲット。現時点では感染台数が限定的だが、拡大の可能性がある

### 地理的分布

| 国 | 感染比率 |
|-----|---------|
| 韓国 | 48.5% |
| 中国 | 31.8% |
| スウェーデン | 6.4% |
| マレーシア | 3.5% |
| シンガポール | 2.5% |

韓国・中国への集中は、これらの地域での D-Link DIR-850L/DIR-818LW の普及度を反映している可能性がある。

### 推奨対応

- **EOL D-Link ルーターを最新機種に交換**（最も効果的な対策）
- ルーターのファームウェアバージョンを確認し、D-Link DIR-850L・DIR-818LW を優先的に交換
- ルーターの管理インターフェースへのインターネット公開を制限
- DNS 設定が改ざんされていないか確認（正規の DNS サーバー IP と一致しているか）
- ルーターを通過するトラフィックの不審な宛先・パターンを監視
