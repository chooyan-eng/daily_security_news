# Synology MailPlus Server 重大脆弱性（CVE-2026-13136 他）— 2100超の公開サーバーに緊急パッチ

- **日付**: 2026-06-29
- **出典**: [Help Net Security](https://www.helpnetsecurity.com/2026/06/26/synology-mailplus-server-vulnerabilities/) / [Synology Security Advisory SA-26:11](https://www.synology.com/en-us/security/advisory/Synology_SA_26_11)
- **トピック**: [Synology MailPlus Server 重大脆弱性（2026年）](../topics/synology-mailplus-server-2026.md)
- **分類**: 新規

## 概要

Synology が6月26日、NAS デバイス上でプライベートメールインフラを運用する「MailPlus Server」に存在する3件の重大な脆弱性を公開した。リモート攻撃者による任意ファイルの読み書きや内部サービスへの不正アクセスが可能となり、Bitsight の調査では 2,100 件超のインターネット公開サーバーが確認されている。即時パッチ適用が必要。

## 詳細

### 脆弱性一覧

| CVE | 深刻度 | 内容 |
|-----|--------|------|
| CVE-2026-13136 | Critical | 任意ファイルの読み書き、およびDoS攻撃が可能 |
| CVE-2026-13135 | Critical | 通信チャンネルの制限不備により、リモート攻撃者が内部サービスへアクセス可能 |
| CVE-2025-15660 | High | 暗号的に脆弱な疑似乱数ジェネレーター（PRNG）の使用により、隣接ネットワーク上の攻撃者が任意ファイルの読み書きおよびDoSを実行可能 |

### 技術的詳細

**CVE-2026-13136**（任意ファイル読み書き）: 攻撃者はリモートからNASデバイス上の任意ファイルにアクセス・変更できる。Synology MailPlus Server が NAS の他の機能とストレージを共有していることから、メール以外のデータも危険にさらされる。

**CVE-2026-13135**（内部サービスへの不正到達）: 通信チャンネルへの意図しない外部アクセスが可能となる設定不備。内部サービスへの SSRF に類似した攻撃経路。

**CVE-2025-15660**（弱い乱数生成）: 暗号的に不十分な疑似乱数ジェネレーターの使用により、セッション・トークン・暗号鍵の予測が可能になる。隣接するネットワーク上の攻撃者が任意ファイル操作やDoSを実行できる。

### 影響範囲と公開サーバー数

Bitsight の Groma Explorer スキャンエンジンが確認した **2,100 件超**のインターネット公開 Synology MailPlus Server の地域分布：

- ドイツ（最多）
- アジア（韓国・中国・台湾）
- 米国

### 対象バージョンと修正

**影響を受けるバージョン**:
- DiskStation Manager (DSM) v7.3、v7.2.2、v7.2.1 上で動作する MailPlus Server

**修正バージョン**:
- MailPlus Server **4.0.1-31663** にアップグレードすることで全3件の脆弱性が修正される

**利用可能な緩和策**: なし（パッチ適用が唯一の対策）

### 攻撃シナリオのリスク

Synology NAS はホームユーザーから中小企業まで幅広く利用されており、MailPlus Server を介した侵害は：
- メールサーバーとしてのデータ窃取（メール内容・添付ファイル）
- NAS 全体へのアクセス拡大（写真・文書・バックアップデータ）
- 内部ネットワークへのピボット

といったリスクをもたらす。
