# DCloud Uni-App が236,000以上のグローバル詐欺サイトに悪用：投資・暗号資産詐欺インフラの実態

- **日付**: 2026-06-27
- **出典**: [SecurityWeek](https://www.securityweek.com/chinese-framework-powers-200000-scam-sites/) / [Infoblox](https://www.infoblox.com/blog/threat-intelligence/from-san-pedro-to-salinas-how-a-chinese-framework-dcloud-uni-app-powers-a-global-scam-economy/)
- **トピック**: [DCloud Uni-App 悪用によるグローバル詐欺インフラ（2026年）](../topics/dcloud-uniapp-scam-infrastructure-2026.md)
- **分類**: 新規

## 概要

中国のオープンソース開発フレームワーク DCloud Uni-App が、世界規模の投資詐欺・暗号資産詐欺・ギャンブル詐欺の基盤として大規模に悪用されていることが Infoblox の調査で明らかになった。2022〜2026年にかけて Uni-App で構築された詐欺ドメインは累計236,000件超に達する。

## 詳細

### DCloud Uni-App とは

DCloud（北京本社）が開発するクロスプラットフォーム開発ツールキット。Vue.js で単一のコードベースを書くことで、モバイルアプリ（iOS/Android）・デスクトップアプリ・モバイル最適化 Web サイトに同時デプロイできる。中国国内で広く普及している正規フレームワークで、**DCloud 自体は詐欺活動に関与していない**。

### 悪用の規模

- **累計詐欺ドメイン**: 236,000件以上（2022〜2026年）
- **ピーク時の新規ドメイン**: 月約15,000件（2024年10月以降に急増）
- **詐欺の種別**: 偽暗号資産取引所、フィッシングサイト、ギャンブルプラットフォーム、投資詐欺（「豚の屠殺」型含む）

### 注目の事例：Lightning Shared Scooter Co.（LSSC）

米国（San Pedro〜Salinas 地区）で確認された詐欺操作。被害者に対して「ハイテク電動スクーターシェアリング会社への出資で高いパッシブ収益が得られる」と偽り、数百万ドル規模の被害をもたらした。Uni-App のクロスプラットフォーム機能を活用し、Web とモバイルアプリの両面から接触する手法を採用。

### 攻撃者が Uni-App を選ぶ理由

1. **単一コードベース**: Web・iOS・Android に同時展開可能 → 詐欺サイトの量産が容易
2. **中国語圏のエコシステム**: テンプレートやコンポーネントの大規模コミュニティ → 詐欺テンプレートの売買市場が形成
3. **正規フレームワークへの隠れ蓑**: ブロックリストへの登録・悪意判定が遅くなる
4. **モバイル親和性**: モバイルアプリとして直接インストールさせることも可能

### フィンガープリント特定の方法

Infoblox は Uni-App 固有の HTML/JS パターン（`window.__uniapp_x_router__` などの変数名）を指紋として詐欺サイトを識別・追跡した。このフィンガープリント手法により約236,000件のドメインを特定。

### モバイルアプリセキュリティへの示唆

Uni-App を使ったモバイルアプリは APK（Android Package）や IPA として配布されることもあり、正規のアプリストア審査をすり抜けてインストールされるケースも確認されている。ユーザーは金融・投資関連のアプリをインストールする際は公式ストア経由・正規企業確認を徹底することが重要。
