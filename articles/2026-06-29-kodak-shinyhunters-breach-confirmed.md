# Kodak、ShinyHuntersによる220万件データ侵害を公式確認 — 身代金期限後にデータ公開の脅迫継続

- **日付**: 2026-06-29
- **出典**: [BleepingComputer](https://www.bleepingcomputer.com/news/security/kodak-confirms-data-breach-claimed-by-shinyhunters-extortion-gang/) / [Malwarebytes](https://www.malwarebytes.com/blog/news/2026/06/kodak-confirms-breach-as-shinyhunters-leak-threat-reaches-deadline) / [Cybernews](https://cybernews.com/security/shinyhunters-claims-kodak-hack-2-million-records/)
- **トピック**: [ShinyHunters による Oracle PeopleSoft 攻撃キャンペーン](../topics/shinyhunters-oracle-peoplesoft.md)
- **分類**: 関連

## 概要

ShinyHunters が6月15日にKodakを恐喝サイトに掲載し、220万件の顧客PII・社内データを窃取したと主張していた件で、Kodak が公式に侵害を確認した。ShinyHunters は身代金の期限（6月18日）を設定し「従わなければデータを公開する」と脅迫していたが、Kodak は被害範囲は限定的と主張している。同グループは2026年単独でCharter Communications（4200万件）・Oracle PeopleSoft（100社超）・Instructure Canvas（9000超教育機関）等への攻撃を主張しており、大企業を標的にした恐喝活動を継続している。

## 詳細

### 侵害の経緯

ShinyHunters は2026年6月15日に自グループのダークウェブ恐喝サイトに Kodak をリスト掲載し、以下を主張した：

- **窃取件数**: 220万件超のレコード
- **含まれるデータ**: 顧客のPII（個人識別情報）および社内データ
- **最終期限**: 2026年6月18日（この日までに連絡なければデータ公開）

Kodak は侵害を確認したが、「被害は限定的で封じ込めた」「システムや業務運営への脅威はない」と発表。ただし、どの顧客データにアクセスされたかの詳細は非公表。

### ShinyHunters の手口

ShinyHunters が2026年に確立した恐喝パターン：

1. 被害企業をダークウェブ公開サイトに掲載
2. 短期間の期限設定（72時間〜1週間程度）
3. 支払いがなければデータを売却/公開すると脅迫
4. メディア露出による企業への圧力増大

**2026年の主要標的**:
| 組織 | 主張された規模 |
|------|--------------|
| Charter Communications | 4,200万件 |
| Oracle PeopleSoft 顧客群 | 100社以上 |
| Instructure Canvas | 9,000超の教育機関 |
| Kodak | 220万件 |

### Oracle PeopleSoft キャンペーンとの関係

現時点で Kodak の侵害が Oracle PeopleSoft 経由であるという証拠はなく、ShinyHunters が異なる手口を使った可能性が高い。ただし、同グループが2026年を通じて複数の大企業を標的にした連続的な恐喝キャンペーンを展開していることは共通しており、攻撃スタイル・恐喝手法・データ窃取戦術は一貫している。

### 被害の潜在的影響

Kodak は写真フィルムの製造で知られるが、現在は商業印刷・産業用コーティング・機能性材料など幅広い分野で事業展開している。

- **顧客PII**: 顧客の連絡先・注文情報等が含まれる可能性
- **社内データ**: 製造ノウハウ・営業情報等の競業上の機密が含まれる可能性
