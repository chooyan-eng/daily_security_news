# ShapedPlugin WordPress プロプラグインのサプライチェーン攻撃（CVE-2026-10735）

- **日付**: 2026-06-23
- **出典**: [The Hacker News](https://thehackernews.com/2026/06/shapedplugin-wordpress-pro-plugins.html)
- **トピック**: [ShapedPlugin WordPress サプライチェーン攻撃（CVE-2026-10735）](../topics/shapedplugin-wordpress-supply-chain-2026.md)
- **分類**: 新規

## 概要

WordPressプラグイン開発会社 ShapedPlugin のビルド・配布パイプラインが侵害され、プレミアムプラグイン更新ファイルにバックドアが混入。40万件以上のアクティブインストールを持つ同社のプロプラグインを通じて、管理者認証情報や2FAシークレットを窃取するマルウェアが配布された。CVE-2026-10735（CVSS 9.8）および CVE-2026-49777（CVSS 10.0）が付番。

## 詳細

### 攻撃の概要

2026年5月21日、攻撃者は ShapedPlugin のビルド・配布パイプラインへの侵害に成功し、プロプラグインのアップデートパッケージに悪意あるコードを混入し始めた。侵害は約3週間にわたって検出されず、2026年6月初旬に顧客からの不審な報告を受けた Wordfence Threat Intelligence チームが調査を開始し、6月11日に公式に開示された。

### 手法

攻撃者は ShapedPlugin が利用していた Easy Digital Downloads（EDD）アップデートシステムを標的とし、プレミアムプラグインの正規アップデートにバックドアコードを差し込んだ。配布されたマルウェアには以下のツールが含まれていた：

- **Tiny File Manager 2.6**：リモートファイル操作機能
- **Adminer 5.2.1**：データベース直接アクセスツール
- **Webシェル**：サーバーへのリモートコマンド実行
- **REST APIバックドア**：永続的アクセスの確保
- **認証情報窃取コンポーネント**：管理者パスワードと2FAシークレットの収集
- **ログインバイパス機構**：認証を迂回する不正アクセス手段

### 影響範囲

侵害されたプラグインはプロ（有料）版のみ3製品：

1. **Product Slider Pro for WooCommerce** - バージョン 3.5.4 未満（CVE-2026-49777、CVSS 10.0）
2. **Real Testimonials Pro** - バージョン 3.2.5
3. **Smart Post Show Pro** - バージョン 4.0.2 未満

ShapedPlugin の無料プラグインには影響がなく、WordPress.org リポジトリ経由の配布も影響を受けていない。同社の無料製品のアクティブインストール数は40万件以上。

### CVE情報

- **CVE-2026-10735**（CVSS 9.8）：サプライチェーン攻撃全体に付番
- **CVE-2026-49777**（CVSS 10.0）：Product Slider Pro for WooCommerce の侵害に特定

### 対応・修正

ShapedPlugin は影響を受けた3製品の修正版をリリースした。WordPress サイト管理者は直ちにアップデートを適用し、侵害の痕跡（バックドアファイルの有無、不審な管理者アカウントの追加など）を確認することが推奨される。管理者パスワードの変更と2FA再設定も必須。

### 背景

WordPress エコシステムにおけるサプライチェーン攻撃は近年増加傾向にある。2026年6月には Awesome Motive の CDN サプライチェーン攻撃（別トピック）も発生しており、プラグイン開発者のビルドパイプラインのセキュリティが改めて問われている。
