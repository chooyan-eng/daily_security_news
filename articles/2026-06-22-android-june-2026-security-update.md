# Google Android 2026年6月セキュリティアップデート続報：CVE-2025-48595の実際の悪用拡大とCISA警告

- **日付**: 2026-06-22
- **出典**: [Aviatrix Threat Research](https://aviatrix.ai/threat-research-center/google-june-2026-android-update-patches-124-flaws-one-actively-exploited/)
- **トピック**: [Android CVE-2025-48595 特権昇格ゼロデイ（2026年6月）](../topics/android-cve-2025-48595-zero-day.md)
- **分類**: 続報

## 概要

2026年6月に公開されたAndroidセキュリティアップデートで修正された CVE-2025-48595（Androidフレームワーク特権昇格ゼロデイ）について、CISAが実際の悪用活動の拡大を確認し連邦機関に対して緊急パッチ適用を求めた。限定的・標的型とされていた悪用が、広範な攻撃キャンペーンに発展しつつある兆候が見られる。

## 詳細

### 背景

2026年6月初旬に公開されたGoogleのAndroidセキュリティアップデートは124件の脆弱性に対処しており、その中でも CVE-2025-48595 は唯一「実際に悪用されている」とGoogleが明示したゼロデイ脆弱性である。

### CVE-2025-48595 の再評価

当初「限定的・標的型の悪用」と評価されていた CVE-2025-48595 について、以下の新たな情報が明らかになった：

**CVSS スコア**: 8.4（High）
**影響バージョン**: Android 14、15、16、16-QPR2
**対象コンポーネント**: Androidフレームワーク（Framework）における整数オーバーフロー（Integer Overflow）
**必要条件**: ローカルアクセス、ユーザー操作不要
**攻撃後の影響**: 特権昇格（一般アプリ権限→システム権限）

### 悪用状況の拡大

6月22日時点での悪用状況の更新情報：

1. **CISA KEV（Known Exploited Vulnerabilities）への追加**: CVE-2025-48595 は CISAのKEVカタログに登録済み
2. **連邦機関への命令**: 米連邦行政府機関（FCEB）に対してパッチ適用期限が設定された
3. **悪用パターンの変化**: 当初の国家支援型攻撃者による標的型攻撃から、商業的なスパイウェア業者・RaaS グループが本脆弱性を組み込み始めている兆候

### 実際の攻撃チェーン

CVE-2025-48595 は単独でも危険だが、他の脆弱性と組み合わせることでより強力な攻撃チェーンが構成される：

- **ブラウザ/アプリ脆弱性（リモート攻撃面）** → マルウェア初期インストール
- **CVE-2025-48595（ローカル特権昇格）** → システム権限取得
- **永続化** → スパイウェア・監視ツールのインストール

### 各メーカーのアップデート状況

Androidのセキュリティアップデートはまず Google Pixel に適用され、その後 Samsung、Xiaomi、OPPO 等のメーカーが順次展開する：

| メーカー | 展開状況（2026年6月22日時点） |
|---------|---------------------------|
| Google Pixel | 配布完了 |
| Samsung | 主要モデルに展開中 |
| 他メーカー | メーカー・モデルによる（数週間〜数ヶ月の遅延が一般的） |

### 推奨対策

1. Android デバイスの設定 > セキュリティ > セキュリティアップデートを確認し、最新のパッチを適用する
2. 自動更新が有効になっていることを確認する
3. 2026年6月パッチレベル（2026-06-05）以降のアップデートが適用されていることを確認する
4. モバイルデバイス管理（MDM）を利用している組織は、管理対象デバイスへのパッチ強制適用ポリシーを設定する
