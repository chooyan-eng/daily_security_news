# FBI・SSU合同警告：ロシア諜報機関がSignalバックアップリカバリーキーを標的に大規模フィッシング

- **日付**: 2026-06-29
- **出典**: [BleepingComputer](https://www.bleepingcomputer.com/news/security/fbi-russian-hackers-now-target-signal-backup-recovery-keys/) / [The Hacker News](https://thehackernews.com/2026/06/fbi-warns-russian-intelligence-hackers.html) / [FBI IC3 Advisory](https://www.fbi.gov/investigate/cyber/alerts/2026/russian-intelligence-services-continue-to-target-commercial-messaging-applications) / [GlobalSecurity.org SSU発表](https://www.globalsecurity.org/intell/library/news/2026/intell-260625-ukraine-ssu01.htm)
- **トピック**: [ロシア諜報機関によるSignal・メッセージングアプリ大規模スパイキャンペーン（2026年）](../topics/russian-signal-backup-phishing-2026.md)
- **分類**: 新規

## 概要

米FBI・ウクライナSBU（保安局）は6月25〜27日にかけて合同で、ロシア連邦保安局（FSB）を含む複数のロシア諜報機関が欧米・ウクライナの政府関係者・軍人・政治家・ジャーナリストを標的に Signal などのメッセージングアプリを侵害する大規模キャンペーンを実施していると警告した。最新の手口では Signal の「バックアップリカバリーキー」を詐取することで、端末を持たなくてもメッセージ履歴全体へのアクセスが可能になる。

## 詳細

### キャンペーンの全体像

SBU（ウクライナ保安局）と FBI は6月25日の合同発表で、複数のロシア諜報機関が少なくとも2026年初頭から以下の人物を標的にした組織的なハッキングキャンペーンを展開していると明らかにした：

- ウクライナ・欧州・米国の現職・元職政府関係者
- 軍人および軍関係者
- 政治家・活動家
- ジャーナリスト・高価値情報保持者

このキャンペーンは2026年3月にオランダ国内情報機関（AIVD）と軍情報機関（MIVD）が最初に公表。政府職員のSignal・WhatsAppアカウントが侵害されていた。

### 最新の攻撃手法：バックアップリカバリーキーの詐取

FBIが6月27日に更新した勧告では、攻撃手法が進化していることが明らかにされた。従来の「ワンタイム認証コード詐取」から、**Signal バックアップリカバリーキー**を直接狙う手口へ変化している。

**攻撃フロー**:
1. 攻撃者は Signal のサポートボットを装ったSMSを送信
2. 「同期の問題でデータが失われる恐れがある」と警告
3. 「データ損失を防ぐためバックアップキーをコピーして返信するよう」要求
4. 被害者がキーを送信すると攻撃者が取得

**バックアップリカバリーキーの危険性**:
- キーがあれば攻撃者は自分の端末にバックアップを復元し、メッセージ履歴全体にアクセス可能
- 同じ電話番号で新しい Signal アカウントを作成しても、盗まれたキーは無効にならない
- プライベートチャットとグループ会話の両方を含む過去の全メッセージが露出する

### 脅威アクターの帰属

FBI は今回の勧告で以下の追跡名を正式に追加した：

| 追跡名 | 属性 |
|--------|------|
| UNC5792 | FSB（ロシア連邦保安局）関連 |
| UNC4221 | FSB 国境警備隊に配属された士官および軍関係者 |
| Star Blizzard | 同様の活動に関与が確認された別グループ |

### 推奨される対策

1. **Signal のバックアップリカバリーキーを誰にも共有しない**（Signal サポートが要求することは絶対にない）
2. Signal の「セーフティナンバー」を接触者と定期的に確認する
3. 不審なメッセージを受け取った場合は速やかに削除し、報告する
4. 可能であれば Signal に加えて別の認証チャンネルで相手を確認する
